## 🛠️ Features
- **Symptom Advisor:** Interactive tool helping patients check common symptoms.
- **Appointment Booking:** Seamless scheduling integration.
- **Responsive Design:** Optimized for mobile, tablet, and desktop viewing.
- **Interactive Animations:** Rich transitions, hover effects, and modern styling.

## 📁 Project Structure
- `index.html` - Core structure and content
- `styles.css` - Custom styling and design system
- `app.js` - Dynamic page transitions, Symptom Advisor logic, and mobile menu handling
- `assets/` - Images and SVG icons used throughout the site

## 🌐 Local Development
To view the website locally, simply open `index.html` in any web browser, or use a local development server like Live Server in VS Code:
```bash
# If using Python
python -m http.server 8000
```
